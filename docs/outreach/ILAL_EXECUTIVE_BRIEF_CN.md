└─────────────────────────────────────────────────────┘

### 核心创新：Session 缓存

**传统方法：**
```
每笔交易：
1. 生成 ZK 证明（4.5 秒）
2. 链上验证（252k Gas）
3. 执行交易

每笔成本：$50-100（高峰期 Gas）
用户体验：糟糕（每次等待 4.5 秒）
```

**ILAL Session 方法：**
```
首次交易：
1. 生成 ZK 证明（4.5 秒）
2. 创建 30 天 Session（54k Gas）
3. 执行交易（8k Gas）

后续 1000 笔交易：
1. 检查 Session（8k Gas）
2. 执行交易（8k Gas）

每笔成本：摊销 $0.37/月
用户体验：即时（类似 Web2）
```

**结果：** 成本降低 5,405 倍，体验提升 100 倍
