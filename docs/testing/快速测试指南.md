# ILAL 快速测试指南（无需 PostgreSQL）

## 当前情况

- ✅ Node.js 已安装 (v24.13.0)
- ✅ npm 已安装 (11.6.2)
- ❌ PostgreSQL 需要 sudo 权限安装
- ❌ pnpm 需要 sudo 权限安装

## 解决方案：使用 npx 和本地安装

不需要全局安装任何工具，不需要 sudo 权限！

### 方式 1: 使用 npx（推荐，最简单）

```bash
cd /Users/ronny/Desktop/ilal

# 直接使用 npx 运行 pnpm（无需安装）
npx pnpm install --prefix apps/api

# 使用 SQLite 代替 PostgreSQL（无需安装数据库）
cd apps/api

# 修改为使用 SQLite
cat > .env << 'EOF'
DATABASE_URL="file:./dev.db"
JWT_SECRET="dev-jwt-secret-for-testing-only"
API_KEY_SECRET="dev-api-key-secret-for-testing"
PORT=3001
NODE_ENV="development"
RPC_URL="https://base-sepolia-rpc.publicnode.com"
CHAIN_ID=84532
VERIFIER_PRIVATE_KEY="0x1234567890123456789012345678901234567890123456789012345678901234"
SESSION_MANAGER_ADDRESS="0x53fA67Dbe5803432Ba8697Ac94C80B601Eb850e2"
VERIFIER_ADDRESS="0x0cDcD82E5efba9De4aCc255402968397F323AFBB"
EOF

# 运行数据库迁移（使用 SQLite）
npx prisma migrate dev --name init

# 启动服务
npx tsx src/index.ts
```

### 方式 2: 本地安装 pnpm

```bash
cd /Users/ronny/Desktop/ilal

# 安装 pnpm 到项目本地（无需 sudo）
npm install pnpm

# 使用本地 pnpm
./node_modules/.bin/pnpm install
```

### 方式 3: 使用 npm 代替 pnpm

```bash
cd /Users/ronny/Desktop/ilal/apps/api
npm install

# 配置环境变量（同上）
# ...

npm run dev
```
